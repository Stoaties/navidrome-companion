Public access

Set a domain that points at this server's public IP. Caddy obtains an HTTPS certificate automatically. Leave blank to serve plain HTTP.

Spotify

No setup needed for most links: paste a public Spotify playlist, album or track on the dashboard and the track list is read from Spotify's public page, then each track's audio is fetched from YouTube and tagged. Private playlists aren't accessible; set them to public first.

The public page only lists the first 100 tracks of a playlist. To import bigger playlists in full, create a free app at developer.spotify.com/dashboard and paste its Client ID and Secret below — they're used read-only to page through the whole list.

Metadata (MusicBrainz)

Look up each downloaded track in the free MusicBrainz database to fill in the canonical track, artist and album name (Spotify playlists don't provide a per-track album) plus MusicBrainz IDs, so Navidrome groups albums correctly and can show artist art. Adds ~1-2s per track (MusicBrainz rate-limits lookups), and only tags an album when it's confidently a studio release. Turn off if you'd rather keep imports fastest.
